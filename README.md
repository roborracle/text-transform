# Text Transform

Developer-focused text and data transformation toolkit with 111+ client-side tools.

## Features

- **Naming Conventions** - camelCase, snake_case, kebab-case, PascalCase, and more
- **Encoding/Decoding** - Base64, URL, HTML entities, binary, hex
- **Cryptography** - MD5, SHA-256, UUID, HMAC, timestamps
- **Code Formatters** - JSON, SQL, CSS, HTML, XML beautify and minify
- **Data Converters** - CSV↔JSON, XML↔JSON, Markdown↔HTML, cURL to code
- **Color Utilities** - HEX, RGB, HSL conversions (linked to ColorCodeGuide.com)
- **Random Generators** - Passwords, UUIDs, Lorem ipsum, test data
- **Ciphers** - Caesar, ROT13, Morse code, Vigenère

## Stack

- Next.js 15 with App Router
- React 19
- TypeScript (strict mode)
- Tailwind CSS 4

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
